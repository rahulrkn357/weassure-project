# WeAssure – TPA Web Application

**Live Demo**
Render Link:  https://weassure-project-1.onrender.com

##Project Overview

**WeAssure** is a TPA (Third Party Administrator) web application designed to streamline insurance workflows and reduce delays in hospital admissions and cashless claim initiation.

##Features

- Developed a **network hospital & blacklist verification system** to instantly validate hospital eligibility, replacing manual insurer checks and reducing verification time by **20%**.  
- Built an **e-card download feature** serving as proof of insurance enrollment, reducing administrative delays by **50%** and enabling faster initiation of cashless treatment requests.  
- Designed a **responsive and user-friendly interface** with **dark mode support** to enhance usability during extended operational use.  

##Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Tailwind CSS  
- **Backend:** Node.js  
- **Database:** MySQL (hosted on Aiven Cloud)  
- **Deployment:** Render
